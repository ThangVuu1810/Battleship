🔥🚢 Battleship 2D - Python Game
Một tựa game chiến thuật Battleship được phát triển bằng Python và Pygame. Dự án áp dụng các nguyên lý lập trình hướng đối tượng (OOP) và tích hợp thuật toán AI dò đường cơ bản.

🎮🤑 Tính năng nổi bật
Chế độ EasyAI(chế độ dễ-bắn ngẫu nhiên) và HardAI(chế độ khó-bắn truy sát)
Giao diện đồ họa và hiệu ứng âm thanh đầy đủ.
Tự động lưu trữ lịch sử trận đấu.

🚀 🎶Hướng dẫn cài đặt
Yêu cầu hệ thống cài đặt Python 3.12.
Cài đặt thư viện: pip install pygame
Chạy game: python main.py
